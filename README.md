# ❄️ Benoît — AI Research Collaborator

Assistente di ricerca AI per la Prof.ssa Maria Rosaria Lancia, Sapienza University of Rome.

## Funzionalità

- 🔬 **Ricerca scientifica** — collaboratore in analisi matematica, frattali, PDE
- 📝 **Peer Review** — valutazione articoli scientifici per riviste
- 📰 **Newsletter** — aggiornamenti mensili per il gruppo di ricerca
- 🎯 **Slide** — presentazioni LaTeX Beamer per conferenze
- 🎙️ **Voce** — input vocale in italiano
- ∂ **LaTeX** — rendering automatico delle equazioni

## Setup

1. Vai su [console.anthropic.com](https://console.anthropic.com)
2. Crea una API Key
3. Apri l'app e inserisci la API Key quando richiesto

## Deploy su Vercel

1. Carica questo repository su GitHub
2. Vai su [vercel.com](https://vercel.com)
3. Connetti il repository GitHub
4. Deploy automatico in 2 minuti

## Uso

- Scrivi nella casella di testo e premi Invio
- Clicca il microfono 🎙️ per parlare in italiano
- Usa i pulsanti rapidi nella schermata iniziale per attivare le modalità
- Le equazioni LaTeX vengono renderizzate automaticamente

## Note tecniche

- La API Key è salvata solo nella sessione del browser
- Nessun dato viene inviato a server esterni oltre ad Anthropic
- Richiede Chrome per il riconoscimento vocale
